* Description
This is a C project generator. I designed it to mimic [[https://github.com/rust-lang/cargo][cargo]], at least
on a surface level.

The default file structure is like so:
| <project name>
 | Makefile
 | src
  | Makefile
  | main.c
 | target
  | debug
   | <project name>
  | release
   | <project name>
* Build instructions
** Requirements
[[https://libgit2.org][libgit2]]
** How to Build
#+BEGIN_SRC sh
$ make release
#+END_SRC
** Installation
Note: Default installation path is ~/.local/bin
#+BEGIN_SRC sh
$ make install
#+END_SRC
** Uninstallation
#+BEGIN_SRC sh
$ make uninstall
#+END_SRC
* Examples
** Creating a new project
#+BEGIN_SRC sh
$ genc init <project name>
#+END_SRC
** Building a project with debug flags
#+BEGIN_SRC sh
$ make
#+END_SRC
** Building a project without debug flags
#+BEGIN_SRC sh
$ make release
#+END_SRC
** Running a debug build
#+BEGIN_SRC sh
$ make run
#+END_SRC
** Running a debug build with commandline arguments
#+BEGIN_SRC sh
$ make run a="arg1 arg2 arg3"
#+END_SRC
** Remove debug and release builds
#+BEGIN_SRC sh
$ make clean
#+END_SRC
